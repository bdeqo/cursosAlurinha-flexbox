# cursosAlurinha-flexbox

Neste projeto eu utilizei a famosa especificação flexible box para posicionar os elementos na página e abordei as diversas propriedades do flexbox.

Utilizei as propriedades do flexbox para substituir float, inline e inline-block e observei os critérios e características da responsividade na construção do site.
